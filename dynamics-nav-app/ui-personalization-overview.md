##  <span data-ttu-id="51246-161"><a name="PersonalizationWinWeb"></a>Utilisation de la personnalisation entre le client Dynamics NAV Windows et le client web</span><span class="sxs-lookup"><span data-stu-id="51246-161"><a name="PersonalizationWinWeb"></a>Working with personalization between the Dynamics NAV Windows and Web client</span></span>
<span data-ttu-id="51246-162">Avant de personnaliser des pages, il est important de comprendre comment fonctionne la personnalisation entre le client Windows et le client web.</span><span class="sxs-lookup"><span data-stu-id="51246-162">Before you start personalizing pages, it is important to understand how the personalization between the Window client and Web client works.</span></span> <span data-ttu-id="51246-163">Si vous n'utilisez jamais qu'un des deux, cette information n'est pas très importante.</span><span class="sxs-lookup"><span data-stu-id="51246-163">If you will only ever use either the Windows client or the Web client, this information is not so relevant.</span></span> <span data-ttu-id="51246-164">Toutefois, elle a de l'importance si vous commencez à personnaliser des pages en utilisant les deux clients, ou que vous passez de l'utilisation du client Windows à l'utilisation permanente du client web.</span><span class="sxs-lookup"><span data-stu-id="51246-164">However, it becomes important if you begin to personalize pages using both clients or when transitioning from using the Windows client to using the Web client permanently.</span></span>  

-   <span data-ttu-id="51246-165">Si vous utilisez le client Windows pour personnaliser une page spécifique dès le début, vous verrez également les modifications de personnalisation de la page dans le [!INCLUDE[nav_web_md](includes/nav_web_md.md)].</span><span class="sxs-lookup"><span data-stu-id="51246-165">If you use the Windows client to personalize a specific page from the start, you will also see the personalization changes to the page in the [!INCLUDE[nav_web_md](includes/nav_web_md.md)] as well.</span></span>
