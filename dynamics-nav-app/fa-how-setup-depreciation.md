    <span data-ttu-id="75bbd-117">**Remarque** : vous pouvez enregistrer les transactions immobilisation dans la fenêtre **Feuille compta. immo.** ou dans la fenêtre **Feuille immo.**, selon que les transactions sont destinées pour des rapports financiers ou pour la gestion interne.</span><span class="sxs-lookup"><span data-stu-id="75bbd-117">**Note**: You can record fixed asset transactions in the **Fixed Asset G/L Journal** window or in the **Fixed Asset Journal** window, depending on whether the transactions are for financial reporting or for internal management.</span></span> <span data-ttu-id="75bbd-118">Procédez comme suit pour définir quel type de journal est utilisé pour les différentes activités immobilisation par défaut.</span><span class="sxs-lookup"><span data-stu-id="75bbd-118">Follow the next step to define which type of journal is used for the different fixed asset activities by default.</span></span>

## <a name="to-assign-a-depreciation-book-to-a-fixed-asset"></a><span data-ttu-id="75bbd-121">Pour attribuer un registre amortissement à une immobilisation</span><span class="sxs-lookup"><span data-stu-id="75bbd-121">To assign a depreciation book to a fixed asset</span></span>
